### List of events

## Events::Business::Created

When a business is created, we broadcast an event. Subscribers will receive 
an event payload containing information about the business.

## Events::Schema::Created

When a Schema is created, we broadcast an event. Subscribers will receive 
an event payload containing information about the schema.

## Events::User::Updated

When a User is updated, including a Team switch, we broadcast an event.
Subscribers will receive an event payload containing information about the User, 
including the new Team ID.
